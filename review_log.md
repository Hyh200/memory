# 年度相册网站审查记录表

## 审查规则

- 每个 task 都必须有一条审查记录。
- 只有 `结论=通过` 的 task 才能提交 Git。
- 提交后必须补充 `提交哈希`。
- 提交哈希只能在提交后产生，因此审查通过记录先随 task 提交，提交后立即补写哈希并保存审查记录更新。
- 如果审查发现问题，记录问题并保持该 task 未提交，直到修复并复审通过。
